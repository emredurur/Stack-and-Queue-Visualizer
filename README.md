
# Stack and Queue Visualization with C# Windows Forms

## Overview

This project was developed to better understand and visualize the working logic of **Stack** and **Queue** data structures using **C# Windows Forms**.

The application simulates process-based operations and demonstrates how data is added and removed in both structures through a simple graphical interface.

## Features

- Stack (LIFO) simulation
- Queue (FIFO) simulation
- Add / Remove operations
- Process visualization
- User-friendly Windows Forms interface
- Educational algorithm demonstration

## Technologies Used

- C#
- Windows Forms
- Object-Oriented Programming

## Purpose of the Project

The main purpose of this project is to reinforce fundamental data structure concepts through practical implementation and visualization.

## What I Learned

- Difference between Stack and Queue structures
- FIFO and LIFO logic
- Basic algorithm visualization
- Event-driven programming with Windows Forms
- Improving problem-solving skills with C#
